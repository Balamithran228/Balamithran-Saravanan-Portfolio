# Balamithran Saravanan - Data Engineering Portfolio

A modern, interactive holographic portfolio website showcasing my experience as an Associate Data Engineer specializing in building scalable data pipelines with Snowflake, BigQuery, dbt, and Python.

## 🚀 Features

- **Interactive Design**: Custom cursor, glitch effects, and smooth animations
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile
- **Modern Tech Stack**: HTML5, CSS3, JavaScript (GSAP, Three.js)
- **Showcase**: Experience, projects, skills, and certifications
- **Terminal Interface**: Interactive terminal for quick info access

## 🌐 Deployment

This portfolio is deployed using GitHub Pages with GitHub Actions CI/CD pipeline.

### Live URL
Visit: `https://balamithran228.github.io/portfolio`

### Manual Deployment

1. **Clone the repository**
   ```bash
   git clone https://github.com/Balamithran228/portfolio.git
   cd portfolio
   ```

2. **View locally** - Open `index.html` in your browser

3. **Deploy to GitHub**
   - Push to main branch
   - GitHub Actions automatically deploys to GitHub Pages
